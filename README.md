Celestium is an AI-powered space learning platform that helps users explore the universe using real NASA data, interactive lessons, quizzes, and missions.

## Problem Statement
Traditional space education is static and theory-heavy. Celestium makes learning astronomy interactive, personalized, and data-driven.

## Key Features
- NASA APOD feed integration
- AI-generated lessons and quizzes
- Personalized learning dashboard
- Mission-based simulations
- User authentication (Firebase)

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Authentication: Firebase
- APIs: NASA APOD
- CI/CD: GitHub Actions (flake8, formatting)

## How to Run Locally
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies
